#### 安装器必须环境：.NET 4.8 Runtime

模组安装器依赖于 .NET 4.8 运行时，如果您希望使用模组安装器，您需要使用微软提供的安装器来完成这一步。

##### 关于安装问题
- 如果您在安装时遇到问题，请先尝试使用**脱机安装程序**进行安装。  
- 如果仍然无法安装，请确认您的Windows版本：
  - Windows专业版和家庭版最低需要版本1607才能安装.NET4.8框架。
  - Windows 10 1507 以及 Windows 10 1511 不支持.NET4.8框架。您首先需要升级到较新的 Windows 10 版本。
