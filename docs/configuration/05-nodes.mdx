---
title: 模组设置 - 节点切换功能
---

翻译模组安装器(工具箱)的设置页面提供了“切换节点”以及“切换API节点”的功能。

请您先不要调整，以默认设置尝试是否能够正常安装模组。**若您能够以默认设置安装模组，则不建议您对此处的设置进行调整**。  
如果您在默认设置下报错，则可按照以下教程尝试切换不同节点，以改善您与零协会提供的服务的**连接性问题**。

## 切换节点 {#change-dl-node}

切换节点，即切换**下载**节点。安装器将会从您所选择的下载节点，**下载或更新模组**的各个组件。  

若您遇到**以下问题**时，可以考虑更换此处的下载节点：  
- 点击“开始安装”后报错 的情况下
- 安装条读完，弹窗显示模组安装成功后，模组实际上却没有被安装 的情况下
- 哈希检测失败 的情况下

请**随意切换下载节点，直到找到一个可以使用的节点**。  
若您发现全部节点都无法使用，请您在重试**一两次**(请**不要**持续重试)之后，暂时放弃使用安装器(此时可以考虑进行[手动安装](https://www.zeroasso.top/docs/install/install))。

不同节点的介绍(若您不确定以下信息代表什么，则可忽略)：
- OnedriveForBusiness - 连接零协会服务器以获取OneDrive For Business上的文件；对国内网络友善；其是默认节点
- Cloudflare CDN - 连接Cloudflare CDN以获取其上缓存的零协会服务器的文件；对国内网络友善
- 天翼云盘 - 连接零协会服务器以获取天翼云盘上的文件；对国内网络友善
- 联通云盘 - 连接零协会服务器以获取联通云盘上的文件；对国内网络友善
- 移动云盘 - 连接零协会服务器以获取移动云盘上的文件；对国内网络友善
- 电信云盘 - 连接零协会服务器以获取电信云盘上的文件；对国内网络友善
- GitHub 直连 - 直接获取GitHub上的文件
- Mirror GitHub - 直接获取GitHub镜像站上的文件；对国内网络友善

## 切换API节点 {#change-api-node}

安装器将会连接您所选择的API节点，以使用一些需要**使用互联网与零协会服务器通信**才能正常运行的功能。这些功能包括：  
- 模组版本检测，检查是否存在新版本
- **不包括**模组文件的下载

若您遇到**以下问题**时，可以考虑更换此处的API节点：  
- 即便存在更高版本可以更新，使用“开始安装”进行更新完毕后，依然是旧版本

请**随意切换API节点，直到找到一个可以使用的节点**。  
若您发现全部节点都无法使用，请您在重试**一两次**(请**不要**持续重试)之后，暂时放弃使用安装器(此时可以考虑进行[手动安装](https://www.zeroasso.top/docs/install/install))。
