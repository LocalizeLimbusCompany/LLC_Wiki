### （一）.NET 8.0 Runtime

在您首次运行模组安装器时，若您的电脑上没有 .NET 8.0 运行时，那么工具箱程序会在**启动之前**出现如下弹窗：

![image](/img/page/installnet8.png)

请您点击弹窗中的按钮，其将会跳转到您的浏览器，并自动开始下载一个 .NET 8.0 运行时的安装向导。  
您也可以在下面的按钮下载安装程序

待下载完成后，请您运行该安装向导，完成 .NET 8.0 运行时的安装，后续即可正常使用工具箱。

### （三）解压 工具箱

如果你能够解压.7z格式的压缩包则可以直接右键.7z并解压至单独文件夹，位置随意，工具箱可以自动寻找到边狱公司的路径。

如果你不能解压.7z，请在 [7zip官网](https://7-zip.org/a/7z2301-x64.exe) 下载7zip软件并安装即可。

### （四）运行 工具箱

运行LLC_MOD_Toolbox.exe，点击一键安装，并等待安装完成后运行游戏即可。
