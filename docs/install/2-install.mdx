如果自动安装出现了问题，则您可以选择手动安装方式。

## （可选）下载 Watt Toolkit（原Steam++）

我们已经在标准安装流程中去除了对GitHub的依赖。不过，若您选择从GitHub获取文件，那么您在安装过程中依然需要访问Github。

如果你有全局代理（梯子）等工具则可以无视，如果你没有，请在 [Watt Toolkit官网](https://steampp.net/) 下载并解压。  
打开后勾选Github，再点击加速。在接下来的安装步骤中就别关上了。

## 下载 .Net 6.0

模组的运行离不开 .Net 6.0 环境。  
请务必进入 [此链接](https://download.visualstudio.microsoft.com/download/pr/4a725ea4-cd2c-4383-9b63-263156d5f042/d973777b32563272b85617105a06d272/dotnet-sdk-6.0.406-win-x64.exe) 下载 .Net 6.0 安装包并运行。

## 下载组件

您需要且**必须**下载以下压缩包。  
注意，如果你使用**Github Release**下载文件，切勿下载**Source code**或**带有OTA的压缩包**，你应该下载类似于**LimbusLocalize\_BIE\_v0.x.x.7z**的文件。

### BepInEx (a.k.a BIE)

:::info

LLC特供版BepInEx预装了IL2CPP的依赖文件，且去除了不需要使用到的输出程序集。因此，其初始化时不需要额外从互联网下载任何文件。  
我们推荐您优先选用我们提供的BepInEx以减少额外下载带来的网络问题。

如果您选择直接从BepInEx获取框架，您应该选择**支持IL2CPP的_Bleeding Edge Build_**，而不是直接从GitHub获取其释放。(2023/8/12)

:::

## 安装 7zip（可选）

如果你下载完刚刚的压缩包，发现你的计算机无法识别.7z文件，则你需要安装7zip。

你可以在 [7-zip 官网](https://7-zip.org/a/7z2301-x64.exe) 下载安装包。

## 解压

打开你的steam，进入库，右键Limbus Company，点击浏览本地文件。

将BepInEx，零协会汉化补丁主体和字体的压缩包拖入这个目录。

将所有压缩包解压，请务必注意：**不要选择新建一个目录的解压，应选择解压至当前文件夹(360压缩) 或 提取到当前位置(7zip)**。  
解压完成后，建议您删除压缩包。

## 完成

您的游戏根目录看起来应该是这样的：

![image](/img/page/BIEfiles.png)

现在点击进入`\BepInEx\plugins`文件夹，应该有三个文件(夹)，分别为：`Localize`、`LimbusLocalize_BIE.dll`、`tmpchinesefont`。  
如果您的游戏文件能够对应以上体系，那么您的汉化不应该出现问题。
