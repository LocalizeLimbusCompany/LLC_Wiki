## 下载组件

您需要下载以下**所有**下压缩包。我们建议您按照 BepInEx → 字体文件 → 模组本体 的顺序进行下载。  
若您使用**Github Release**下载文件，切勿下载**Source code**或**带有OTA的压缩包**。您应下载类似于**LimbusLocalize\_BIE\_v0.x.x.7z**的文件。

### 一、BepInEx (a.k.a BIE)
:::info

LLC特供版BepInEx预装了IL2CPP的依赖文件，且去除了不需要使用到的输出程序集。因此，其初始化时不需要额外从互联网下载任何文件。  
我们推荐您优先选用我们提供的BepInEx以减少额外下载带来的网络问题。

如果您选择直接从BepInEx获取框架，您应该选择**支持IL2CPP的_Bleeding Edge Build_**，而不是直接从GitHub获取其释放。(2023/8/12)

:::

## 使用 7zip（可选）

如果你下载完刚刚的压缩包，发现你的计算机无法识别.7z文件，则建议您使用7zip。  
可以在 [7-zip 官网](https://7-zip.org/a/7z2301-x64.exe) 下载安装包。

## 解压

打开steam，进入库，右击Limbus Company，在“属性”选项内，点击浏览本地文件。

将BepInEx，零协会汉化补丁主体和字体的压缩包拖入这个目录。

将所有压缩包解压，请务必注意：**不要选择新建一个目录的解压，应选择解压至当前文件夹(360压缩) 或 提取到当前位置(7zip)**。  
解压完成后，建议您删除压缩包。

## 完成

您的游戏根目录看起来应该是这样的：

![image](/img/page/BIEfiles.png)

现在点击进入`\BepInEx\plugins`文件夹，应该有三个文件(夹)，分别为：`Localize`、`LimbusLocalize_BIE.dll`、`tmpchinesefont`。  
如果您的文件能够对应以上体系，那么您的汉化不应该出现问题。
