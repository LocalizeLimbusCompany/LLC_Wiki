## 使用工具箱卸载汉化
[V0.4.0版本](/docs/install/autoinstall/#%E4%BA%8C%E4%B8%8B%E8%BD%BD-%E5%B7%A5%E5%85%B7%E7%AE%B1)(蓝绿色)及以上的安装工具箱均内置了删除汉化补丁的功能。  
您可以通过**设置页面上的**该功能一键卸载汉化模组，以方便您重装或使用原版游戏。

## 手动卸载汉化

您可以至游戏根目录手动删除`winhttp.dll`, `doorstop_config.ini`, `BepInEx`, `dotnet` 四个文件或文件夹，即可完全删除汉化模组。

若您曾下载过工具箱，将其一并删除即可。
