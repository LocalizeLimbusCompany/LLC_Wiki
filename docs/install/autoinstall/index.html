推荐-自动安装

警告

汉化模组安装器(即工具箱)仅支持Windows 10 及以上版本。
汉化模组本体不支持 Windows 7，在 Windows 8 上未被验证。

我们推荐您使用模组安装器(即工具箱)进行自动安装！如若您对自己的动手能力有信心，您也可以选用手动安装。

优点：​

  • 可选不同下载节点，同时也能选择手动开启从Github下载
  • 一键安装，打开软件后仅需点击一键安装后等待安装完成即可完成安装
  • 全程无需访问Github，纯国内网络即可！

安装流程​

注意

使用模组安装工具箱之前，请您务必先通读本篇安装流程。
若您遇到安装问题，您可以先至常见问题页面寻找可能的解决方法。

（一）.NET 8.0 Runtime​

在您首次运行模组安装器时，若您的电脑上没有 .NET 8.0 运行时，那么工具箱程序会在启动之前出现如下弹窗：

请您点击弹窗中的按钮，其将会跳转到您的浏览器，并自动开始下载一个 .NET 8.0 运行时的安装向导。
您也可以在下面的按钮下载安装程序

.NET 8.0 运行时程序

待下载完成后，请您运行该安装向导，完成 .NET 8.0 运行时的安装，后续即可正常使用工具箱。

（二）下载 工具箱​

在下方选择一个节点下载工具箱。

OneDrive For BusinessGithub Release

（三）解压 工具箱​

如果你能够解压.7z格式的压缩包则可以直接右键.7z并解压至单独文件夹，位置随意，工具箱可以自动寻找到边狱公司的路径。

如果你不能解压.7z，请在 7zip官网 下载7zip软件并安装即可。

（四）运行 工具箱​

运行LLC_MOD_Toolbox.exe，点击一键安装，并等待安装完成后运行游戏即可。

出现问题？​

  • 请优先参考常见问题页面中的提示。
  • 向我们反馈问题。
