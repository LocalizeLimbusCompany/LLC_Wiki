手动安装

警告

汉化模组安装器(即工具箱)仅支持Windows 10 及以上版本。
汉化模组本体不支持 Windows 7，在 Windows 8 上未被验证。

提示

手动安装太复杂了？点此尝试 文件覆盖 吧！文件覆盖安装的效果与手动安装相同，并且能解决一些初始化时可能存在的问题。

如果自动安装出现了问题，则您可以选择手动安装方式。

可选 - 下载 Watt Toolkit 或 Steamcommunity 302​

我们已经在标准安装流程中去除了对GitHub的依赖。不过，若您选择从GitHub获取文件，那么您在安装过程中依然需要访问Github。
如果您可以解决连接性问题，则可以跳过此步。

您可在 羽翼城个人博客 下载 Steamcommunity 302。
您可以在 Watt Toolkit网址 下载 Watt Toolkit。

请注意，Watt Toolkit内存在广告投放。
Watt Toolkit操作相对直观，而Steamcommunity 302则更加简约静默。
关于这些工具的其他用法，推荐您自行了解。
以上工具均由其制作者免费提供，请您支持他们。
零协会与以上工具的制作和发行方均无任何关联。

这两个工具主要是为加速Steam而制作的，但也可以加速GitHub。
您需要在其设置页面勾选Github，再开启加速服务。

下载组件​

您需要下载以下所有下压缩包。我们建议您按照 BepInEx → 字体文件 → 模组本体 的顺序进行下载。
若您使用Github Release下载文件，切勿下载Source code或带有OTA的压缩包。您应下载类似于LimbusLocalize_BIE_v0.x.x.7z的文件。

一、BepInEx (a.k.a BIE)​

信息

LLC特供版BepInEx预装了IL2CPP的依赖文件，且去除了不需要使用到的输出程序集。因此，其初始化时不需要额外从互联网下载任何文件。
我们推荐您优先选用我们提供的BepInEx以减少额外下载带来的网络问题。

如果您选择直接从BepInEx获取框架，您应该选择支持IL2CPP的_Bleeding Edge Build_，而不是直接从GitHub获取其释放。(2023/8/12)

OneDrive For BusinessGithub Release

二、字体文件​

OneDrive For BusinessGithub Release

注：若从GitHub下载，您需要自行找到该文件应放置的位置，而不是直接解压至根目录。

三、模组本体​

OneDrive For BusinessGithub Release

注：若从GitHub下载，请选择BIE版全量包：文件名类似LimbusLocalize_BIE_vx.x.x.7z

使用 7zip（可选）​

如果你下载完刚刚的压缩包，发现你的计算机无法识别.7z文件，则建议您使用7zip。
可以在 7-zip 官网 下载安装包。

解压​

打开steam，进入库，右击Limbus Company，在“属性”选项内，点击浏览本地文件。

将BepInEx，零协会汉化补丁主体和字体的压缩包拖入这个目录。

将所有压缩包解压，请务必注意：不要选择新建一个目录的解压，应选择解压至当前文件夹(360压缩) 或 提取到当前位置(7zip)。
解压完成后，建议您删除压缩包。

完成​

您的游戏根目录看起来应该是这样的：

现在点击进入\BepInEx\plugins文件夹，应该有三个文件(夹)，分别为：Localize、LimbusLocalize_BIE.dll、tmpchinesefont。
如果您的文件能够对应以上体系，那么您的汉化不应该出现问题。

仍然出现(无法解决的)问题？​

  • 请优先参考常见问题页面中的提示。
  • 或者，向我们反馈问题以第一时间得到支持。

关于任何 .Net 运行时​

模组本体无需使用任何 .Net 运行时。模组安装器仍需运行时，详情请参照自动安装页面。
