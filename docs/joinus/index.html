翻译&校对​

信息

当前，翻译与校对的入组题目已经合并统一。
若您希望成为零协会翻译或校对，请您翻译我们提供的原文，并告知对应审核老师您希望成为翻译还是校对。

各个语言的翻译是零协会最重要的组成部分，是参与日常翻译活动的主力。零协会翻译模组因此存在。
我们对校对的准确性和遣词造句要求更高。

提示

在翻译过程中，请注意保留其中的换行符\n；使用适合中文的全角符号；使用适合中文的全角引号而不是方引号；不要破坏Json原有格式；请保证您的符号使用符合我们的符号规范。

请您点此下载零协会试译文本，并在该pdf文件的文字域内完成您的翻译编辑，保存后按文件内指示提交。

字幕(打轴)​

信息

字幕任务比较固定，主要是协助对月亮计划视频进行压制。当前，此类工作往往需要您使用类似Pr制作(简单仿月计样式的)文字，以及(偶尔需要的)特效字幕。
考虑到以上情况，我们对传统打轴和压制的需求目前较少。

视频相关(广播局)成员日常工作可能比较分散，但我们理解其单次任务量较高、且在单项任务上难以和其他人分享(与翻译相比)。
广播局组织较为自由，您也可以在这里提出您自己的项目，并和大家一起完成它！

当前，我们在剪辑和剪辑相关工作，需要更多人手来分担任务！很感谢您愿意加入我们！

提示
  • 我们明白此类职位在审核时需要进行的事项较多，因此在您进行本项审核之前，建议您先单独联系审核群内对应审核老师。
  • 如果您先前有其他作品，也可以一并提供~

您可以使用您习惯使用的字幕工具，联系群内剪辑审核老师即可~

文案​

信息

文案是视频项目的源头，是重创作类的任务。我们在新闻类节目、考据、PV等项目均有涉及，因此很欢迎您来创作各种各样(乃至其他类型)的节目。

视频相关(广播局)成员日常工作可能比较分散，但我们理解其单次任务量较高、且在单项任务上难以和其他人分享(与翻译相比)。
广播局组织较为自由，您也可以在这里提出您自己的项目，并和大家一起完成它！

当前，文案相关人员较少，欢迎您来分享您的灵感！

提示
  • 我们明白此类职位在审核时需要进行的事项较多，因此在您进行本项审核之前，建议您先单独联系审核群内对应审核老师。
  • 如果您先前有其他作品，也可以一并提供~

如果您没有其他过往文案撰写或者视频企划案例，您可以使用以下相关素材，根据月计公告内容制作一份月报文案。

我们建议您从以下列出公告中进行选题。(您仅需制作三条内容)：

  • 8月10日前信息
    • Zwei协会南部4科 浮士德&格里高尔 人格技能翻译前瞻
    • 关于游戏内部问题反馈方案的介绍
    • 边狱公司8月10日预定更新 与 延期通知
    • 至8/10定期维护版本为止已确认的问题介绍
  • 8月17日前信息
    • 有关优化工作的计划以及介绍
    • UI/UX改进介绍以及内容列表
    • 8月17日预定更新通知

需包含以下信息：

  • 月报内容
  • 效果/特效需求
  • 所需展示的内容素材(新闻的图片、视频等载体皆可)
  • 主持人A和主持人B的分工

要求：

  • 仅需制作三条内容，不包含开场白和谢幕语。
  • 新闻与新闻之间有一定联系与过渡
  • 风格幽默、诙谐
  • 以自己的手法编写，表现自我风格，展露自我风采
  • 除素材外内容请以word中表格呈现；结束后请将素材与word打包压缩，发送至相应审核老师
