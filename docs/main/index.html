如何联系我们？​

简介​

欢迎来到LLC主页/文档站！ LLC stands for Localize Limbus Company.

在这里，您可以找到我们收集的关于风格、翻译以及工作流程的信息。

请您先阅读本页内容后，使用左侧的边栏或左上角三横线导航到您要去的地方。

为了让珍贵的东西不被忘记，这将是记载并收集我们的思考、思想和智慧的地方。

加入周边社区或加入零协会​

在这里找到如何加入我们的社区(QQ频道、群)或者加入我们！

Details

如果您希望加入翻译组或广播局与我们一起工作，请参考“加入都市零协会”页面~

如果您希望与他人聊天或找到我们，可以加入我们的QQ频道：
QQ频道链接：加入LLC频道
注：频道现在支持像群聊一样出现在主页消息列表里。从设置页面打开“添加频道至消息列表”即可，非常方便。我们已在频道的“驾驶室|零协公告”页面提供了此设置的图像指引，供您参考。

我们采用频道是因为聊天群已经爆满，即使我们每天都清除非活跃人员，单个群最终还是无法容纳所有人。我们既不愿意进行分流(即开二群)，也不愿意再移除发言相对活跃的小伙伴们，因此我们诚邀您加入LLC频道！
